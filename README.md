# 🛫 MeetPilot — Enterprise-Grade Appointment Scheduling

MeetPilot is a premium, full-scale appointment booking platform designed for high-performance teams. Built with a focus on **visual excellence**, **real-time interactivity**, and **bulletproof reliability**, it offers a seamless bridge between your calendar and your clients.

---

## 💎 Premium Design Philosophy
MeetPilot isn't just a tool; it's an experience.
- **Vibrant SaaS Aesthetics**: A meticulously crafted UI using curated HSL color palettes.
- **Dual-Mode Mastery**: Seamless transition between sophisticated Light and surgical Dark modes.
- **Tactile Interactivity**: Smooth Framer Motion transitions and micro-animations that make the app feel alive.
- **Glassmorphic Booking Engine**: A stunning public-facing interface designed to convert visitors into appointments.

---

## 🚀 Key Features

### 📅 Advanced Meeting Orchestration
- **Dynamic Event Types**: Configure 15, 30, or 60-minute sessions with custom buffers and notice periods.
- **Smart Link Sharing**: High-contrast, copyable booking links for instant distribution.
- **Location Flexibility**: Native support for Google Meet, Zoom, MS Teams, and Phone sessions.

### ⚙️ Intelligence-Driven Scheduling
- **Recurring Availability**: Define your weekly rhythm with a single click.
- **Precision Overrides**: Block out dates or set custom hours for specific days without breaking your routine.
- **Timezone Synchronization**: Global-ready scheduling that automatically adjusts to your guest's local time.

### 📊 Tactical Dashboard & Analytics
- **Live Performance Metrics**: Track booking trends, conversion rates, and volume growth.
- **Visual Engagement Heatmaps**: Understand your peak booking times through dynamic Recharts visualizations.
- **Real-time Notifications**: Instant feedback engine for status changes, new bookings, and profile updates.

### 🔐 Enterprise Infrastructure
- **Mock Database Architecture**: A robust `localStorage`-based service layer simulating high-concurrency CRUD operations.
- **Session Persistence**: JWT-simulated authentication with persistent login states.
- **Audit Logs**: Full history of system-generated communications and email simulations.

## 🚦 Getting Started

1. **Clone the Pilot Deck**
   ```bash
   git clone <repository-url>
   ```

2. **Initialize Dependencies**
   ```bash
   npm install
   ```

3. **Ignite the Dev Server**
   ```bash
   npm run dev
   ```

4. **Launch**
   Open `http://localhost:5173` and start scheduling your future.
